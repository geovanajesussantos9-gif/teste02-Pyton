# Simulador de IMC (Streamlit)

Aplicação simples em Streamlit para calcular o Índice de Massa Corporal (IMC), com histórico e download em CSV.

Requisitos
- Python 3.8+

Instalação

```bash
python -m pip install -r requirements.txt
```

Execução

```bash
streamlit run app.py
```

Observações
- O app suporta unidades métricas e imperiais.
- Os cálculos são simples e não substituem avaliação médica.
